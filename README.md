# Practic_FizTech
Репозиторий практики Создание базы данных для управоение компанией продукции
1.Скачивание клонирование репозитория
  1.1. Скопируйте ссылку https://github.com/SuperBacbka/Practic_FizTech.git
  1.2. Склонируйте репозиторий на свой компьютер при помощи Git Bash тспользуя команду git lone https://github.com/SuperBacbka/Practic_FizTech.git
   
2.Для запуска приложения необходима операционная система Windows 10/11 и любая имеющая у вас IDE используйте становщик PIP

3.Установка зависемостей и библиотек 
  3.1 используйте становщик PIP
  3.2 sqllite является встроенной библиотекой и загрузка не требуется 
  3.3 Для установки streamlit откройте терминал от имени адинистратора введите команду pip install streamlit
  
4. Запуск приложения   
  4.1 Для запуска приложения откройте терминал от имени администратора 
  4.2 В терминале пропишите команду streamlit run main.py. 
  4.3 IDE запустит установленный по умолчанию браузер с начальной страницей
  4.4 В случае если браузер не запустится автоматически. запустит браузер в ручную, прописать в адресной строке Local URL: http://localhost:8501        или же ваш локальный адрес
