# Practic_FizTech
Репозиторий практики Создание базы данных для управоение компанией продукции
для запуска приложения необходимо открыть файл main.py в любой IDE 
в терминале прописать команду streamlit run main.py. IDE запустит установленный по умолчанию браузер с начальной страницей
в случае если браузер не запустится автоматически.запустит браузер в ручную, прописать в адресной строке Local URL: http://localhost:8501 или же ваш локальный адрес
